\# better34 🟢



A powerful browser extension to optimize browsing, filtering, and content collection on Rule34. 

\*Assisted by Gemini\* 🤖



\## 🇺🇸 



\*\*better34\*\* turns the standard media browsing experience into a seamless conveyor belt. It automatically tracks your history, manages Telegram uploads, and handles tedious errors for you.



\### ✨ Key Features:

\* \*\*One-Click Telegram Export:\*\* Send Photos, Videos, and GIFs directly to your Telegram channel. The extension automatically detects the media type.

\* \*\*Smart Fallback System:\*\* If a file is too large for Telegram (e.g., >50MB) or has extreme proportions (`invalid\_dimensions`), the extension will gracefully catch the error and automatically download the file to your PC instead.

\* \*\*Flood-Control Handling:\*\* Hits a Telegram rate limit? The button turns into a live countdown timer showing exactly how many seconds you need to wait.

\* \*\*Visual Feed Markers:\*\* Never open the same post twice. The extension adds badges to thumbnails in the search feed: 

&#x20; \* 👁️ — Viewed

&#x20; \* ✅ — Saved to collection (PC or Telegram)

\* \*\*Behavioral Analytics ("Boring Artist"):\*\* If you view 5 posts from the same artist consecutively without saving anything, the extension will suggest leaving this author to save your time.

\* \*\*Auto-Navigation:\*\* Automatically returns you to the search page immediately after successfully saving or sending a post.

\* \*\*UI Tweaks:\*\* A handy "Fit to screen" button for viewing long, unreadable comic strips.

\* \*\*Settings \& Stats Popup:\*\* A multilingual (EN/RU) popup menu to configure your Telegram Bot API \& Chat ID, track download statistics, and view built-in error logs.

\## 🇷🇺 



\*\*better34\*\* — это мощное расширение, которое превращает просмотр и сбор контента на Rule34 в удобный и быстрый конвейер. Оно само следит за историей, отправляет медиа в Telegram и решает проблемы с лимитами.



\### ✨ Главные фичи:

\* \*\*Отправка в Telegram в один клик:\*\* Прямая выгрузка фото, видео и GIF в ваш ТГ-канал с автоматическим определением формата.

\* \*\*Умный обход лимитов:\*\* Если файл слишком большой для Telegram (больше 50МБ) или имеет слишком вытянутые пропорции (`invalid\_dimensions`), расширение само поймет это и скачает файл вам на ПК.

\* \*\*Таймер Флуд-контроля:\*\* Если вы отправляете посты слишком быстро, кнопка превратится в живой таймер обратного отсчета, показывая, сколько секунд нужно подождать до снятия блока Telegram.

\* \*\*Визуальные метки в ленте:\*\* Больше никаких дубликатов. На превью картинок в поиске появляются значки:

&#x20; \* 👁️ — Просмотрено

&#x20; \* ✅ — Сохранено в коллекцию (на ПК или в ТГ)

\* \*\*Аналитика "Скучный автор":\*\* Если вы пролистали 5 постов одного художника подряд и ничего не сохранили, скрипт выведет плашку с предложением поискать кого-нибудь другого, чтобы не тратить время.

\* \*\*Авто-навигация:\*\* Скрипт сам возвращает вас на страницу поиска сразу после успешного сохранения или отправки поста.

\* \*\*Улучшение интерфейса:\*\* Кнопка "Вписать в экран", чтобы удобно читать длинные вертикальные комиксы.

\* \*\*Всплывающее меню настроек:\*\* Меню с поддержкой двух языков (RU/EN), где можно безопасно указать токены Telegram API, посмотреть статистику скачиваний и логи ошибок.

\### ⚙️ How to install / Как установить:

1\. Download the repository / Скачайте репозиторий.

2\. Open your browser extensions page (`chrome://extensions/`) / Откройте страницу расширений.

3\. Enable "Developer mode" / Включите "Режим разработчика".

4\. Click "Load unpacked" and select the extension folder / Нажмите "Загрузить распакованное" и выберите папку.

5\. Open the extension popup, switch to ⚙️ Settings, and paste your Telegram Bot Token and Chat ID / Откройте расширение, зайдите в ⚙️ Настройки и укажите ключи Telegram.
